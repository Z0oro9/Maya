# APKTool Operations Guide

1. Run `apktool_decompile` and `jadx_decompile` for complementary views.
2. Use `analyze_manifest` for exported components and permission mapping.
3. Use `search_decompiled_code` to locate auth, crypto, storage, and API artifacts.
