# iOS Toolchain Guide

1. Use SSH-based wrappers for dump and keychain actions when companion coverage is limited.
2. Validate device state and jailbreak capabilities before assuming filesystem reach.
3. Favor reproducible command logs for class-dump and binary extraction steps.
